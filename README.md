## Objective

Analyze pre-owned cars dataset to identify key factors influencing vehicle resale pricing such as age, mileage, fuel type, engine size and ownership history.

## Data Cleaning

Dropped highly incomplete columns
Filled engine capacity using median
Normalized text fields
Removed duplicates
Standardized category formatting

## Feature Engineering

Created new columns:
car_age
cost_per_km
engine_category
owner_count

## Encoding & Scaling

One-hot encoding of categorical variables
MinMax scaling for numerical features

## Visualizations

Scatter plots
Box plot
Histograms
Bar graphs
Pivot tables

## üèÅ Final Insights

- Newer cars are consistently priced higher
- Low mileage = higher resale value
- Diesel vehicles retain greater price value
- Premium engine cars command more price
- First-owner vehicles priced highest
